Hypertext Markup Language (HTML) is the standard markup languge for creating web pages and web applications. web browsers receive HTML documents from a web server or from local storage and render tnem into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elemets are the building blocks of HTML pages. with HTML constructs, images and other objects, such as interactive forms, may be embeded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as interactive headings ,paragraphes, lists, links, quotes and other items.
